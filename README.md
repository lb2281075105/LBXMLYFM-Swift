# LBXMLYFM-Swift
Swift5项目仿写喜马拉雅App，采用MVC+MVVM设计模式，Moya+SwiftyJSON+HandyJSON网络框架和数据解析。数据来源抓包及部分本地json文件

喜马拉雅App:主要分为 首页 我听 发现 我的 四个大模块,详情可以查看截图
